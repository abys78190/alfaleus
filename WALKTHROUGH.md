# Walkthrough: Lead Enrichment & Qualification Pipeline

This walkthrough demonstrates the end-to-end functionality of the **Alfaleus Lead Enrichment Platform**, fulfilling the assignment's requirement for a 3-minute video or written walkthrough.

## 1. CSV Upload & Validation

1. Navigate to the live web application: [https://frontend-psi-teal-lcd2aq9ol8.vercel.app](https://frontend-psi-teal-lcd2aq9ol8.vercel.app)
2. Click **Upload CSV** in the sidebar.
3. Select a CSV containing leads (must have `name` and `company` OR just a `domain`).
4. The client-side validation instantly reads the file and displays a beautiful table preview of the **first 5 rows**.
5. If the format is wrong, an error banner appears. If valid, click **Enrich Leads**.

## 2. Background Enrichment Pipeline (Railway)

1. Navigate to the **Pipeline** page (the pipeline status view).
2. You will see leads in the `enriching` state.
3. Behind the scenes on the Railway backend, the Celery worker triggers:
   * **Company Website Scraper**: Pulls the raw HTML, extracting text.
   * **LinkedIn Scraper**: Safely scrapes the public profile/company page.
   * **Google News RSS**: Looks for recent funding or product launches.
4. If a single source (like LinkedIn) fails or hits a rate limit, the pipeline continues gracefully, using data from the website and news to complete the enrichment.

## 3. Semantic ICP Scoring & Buying Signals

1. Navigate to the **Dashboard** and wait for a lead to move from `enriching` to `enriched`.
2. The Dashboard table dynamically displays the **Name**, **Company**, **ICP Score** (as a visual progress bar), and **Top Signal**.
3. Click on the lead row to open the **Lead Detail View**.
4. In the Lead Detail View, you will see:
   * **Company Profile & Contact**: The raw enriched fields (Industry, Tech Stack, Size, Role).
   * **ICP Score Breakdown**: The 0-100 score driven by `sentence-transformers` semantic similarity matching (e.g., matching "40 engineers" to "20-100 employees").
   * **Buying Signals**: Specific triggers like "Recent Funding" or "Hiring Expansion" pulled directly from news or LinkedIn data.

## 4. LLM Personalised Outreach Drafts

1. Scroll down in the Lead Detail View to the **Outreach Drafts** section.
2. You will see variants of an email generated by the local **TinyLlama** model running CPU-bound within Railway's memory limits.
3. Notice that the drafts are highly specific, referencing actual data points extracted during enrichment (e.g., "I noticed your recent Series A funding announcement" rather than generic filler).
4. Click the **Copy** button to copy a draft instantly.

## 5. Chrome Extension Real-time Enrichment

1. Unzip the `alfaleus-chrome-extension.zip` file located in the root of the GitHub repository.
2. Load it unpacked into Chrome via `chrome://extensions/`.
3. Open a real LinkedIn profile (e.g., [linkedin.com/in/sundarpichai](https://linkedin.com/in/sundarpichai)).
4. Click the Alfaleus extension icon. It automatically extracts the DOM data without any API calls.
5. Click **Enrich Lead**. The extension bypasses CORS issues using a background service worker to send the data securely to the Railway backend.
6. A spinner runs while the pipeline executes. Once complete, the popup displays the final ICP score and top signal natively.
7. Click **Open Dashboard** from the popup to jump straight into the full web UI for that specific lead.

## 6. CRM Sync

1. In the Lead Detail View, locate the CRM sync badge.
2. It clearly indicates the sync status (e.g., `Synced`, `Pending`, `Failed`).
3. Behind the scenes, the backend queries Notion via its API, searching for an existing domain to deduplicate and update the record rather than creating a duplicate entry.
